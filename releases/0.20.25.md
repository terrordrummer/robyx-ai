## Documentation: agentic loop and `/loop` trigger

Companion to v0.20.24 (which added the recognition logic to agent
prompts). This release updates the user-facing documentation to match.

### Changed

- **``docs/scheduler.md``** -- new "Starting a continuous task" subsection
  under the Continuous Tasks heading. Documents the two activation modes:
  explicit (`/loop`) and conversational (agent suggests proactively).

- **``docs/architecture.md``** -- added a paragraph to the Workspace
  Agents section explaining the agentic loop mechanism and linking to the
  scheduler docs.

### Migration

None. ``bot/migrations/v0_20_25.py`` is a no-op. Docs-only change.
