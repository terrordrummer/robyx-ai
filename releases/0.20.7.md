## Early Typing Indicator in Message Handler

Added `send_typing` call in `handle_message` immediately before `_route_and_process`,
matching the pattern already used by `handle_voice`. This provides instant "typing..."
feedback before the agent is even resolved, complementing the continuous typing loop
added in v0.20.6.
