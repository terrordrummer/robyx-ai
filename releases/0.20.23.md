## Strip TTS summary blocks from outgoing messages

AI agents sometimes append a ``[TTS_SUMMARY]...[/TTS_SUMMARY]`` block at
the end of their response -- a text-to-speech-friendly recap of the same
content. In a chat context this is redundant noise, so the bot now strips
these blocks before delivering the message to the platform.

### Changed

- **``bot/ai_invoke.py``** -- new ``TTS_SUMMARY_PATTERN`` regex.
- **``bot/handlers.py``** -- the response pipeline in
  ``_process_and_send`` strips TTS summary blocks after all other
  pattern processing and before ``_send_response``. Consecutive blank
  lines left by the removal are collapsed.

### Migration

None. ``bot/migrations/v0_20_23.py`` is a no-op. No data/state changes.
