- Immediate typing indicator in supergroup topics
- Multi-workspace and multi-specialist creation in a single response

- Hourly background update check with one-time-per-version notifications

- Robust error handling — errors always produce a user-visible response
- Improved uninstall scripts with full process and state cleanup
- AI-assisted install instructions in README
- Hosted on GitHub
