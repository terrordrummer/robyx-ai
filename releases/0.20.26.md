# Release 0.20.26

**Collaborative Workspaces**

min_compatible: 0.20.0
breaking: false

## Summary

Adds collaborative workspaces -- a new feature that lets external collaborators join a separate Telegram group with a dedicated AI agent. Supports role-based authorization (Owner/Operator/Participant), two interaction modes (intelligent/passive), and in-group lifecycle commands.

## What's New

- **Collaborative workspace data model** with `CollabStore` persistence
- **Role-based authorization**: Owner (full control), Operator (executive instructions), Participant (conversation only)
- **Interaction modes**: Intelligent (agent decides when to respond) and Passive (respond only on @mention)
- **Two creation flows**: Flow A (planned via orchestrator) and Flow B (ad-hoc bot-added-to-group)
- **In-group commands**: `/promote`, `/demote`, `/role`, `/mode`, `/close`
- **Collaborative agent prompt** with multi-user awareness and `[SILENT]` suppression
- **Shared memory**: collaborative workspaces sharing the same `work_dir` automatically share memory

## Migration

No data migration required.
