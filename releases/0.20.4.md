## Bare "help" Command and Improved Help Text

- **Bare `help` in Headquarters** — typing `help` without the `/` prefix
  is now intercepted and handled as `/help`. Small convenience for users
  who forget the slash.
- **Help text refresh** — clearer descriptions for all available commands
  so the output is actually useful to someone seeing the bot for the
  first time.
