## Fix Typing Indicator in Headquarters

`send_typing` now uses a direct httpx API call instead of PTB's `send_chat_action`,
which was unreliable in forum supergroups (same issue that led `send_message` to be
migrated to httpx). For the General topic (headquarters), `message_thread_id` is
omitted entirely instead of passing `0`, which caused the typing indicator to
silently fail.
